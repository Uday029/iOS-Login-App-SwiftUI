# iOS-Login-App-SwiftUI
A simple iOS login application built with SwiftUI using MVVM architecture, API integration, and localization support.
📱 SwiftUI Login App (MVVM)

A modern iOS login application built using SwiftUI and MVVM architecture.
This project demonstrates clean architecture, API handling, and localization in a beginner-friendly way.

---

🚀 Features

- 🔐 Login screen with username & password
- 🧠 MVVM architecture (clean separation of logic)
- 🌐 API integration using URLSession
- 🌍 Localization support (multi-language)
- 🧹 Code quality with SwiftLint
- 📦 Dependency management using CocoaPods

---

🛠️ Tech Stack

- SwiftUI
- MVVM Architecture
- URLSession (Networking)
- CocoaPods
- SwiftLint
